Structuring the Project

It is the starting of the project where I started with designing the navbar for the Portfolio project. I selected the color yellow for background and white as text color. The nav items css are provided with flex display and has been distributed evenly using space-even css to make sure they are seperated evenly across screen which I later changed so that it does not look odd in larger screens

Applying Animation for my name

This has been the challenging part for all I've done from starting the project. Adding animation I have been searching through the web and trying to understand the approach they have taken to accomplish the task. I accomplished this animation using animation css, where I used transform, filter and opacity for the animation effect where every letters are looped for 100 milliseconds for infinite amount of time.

Working with Contact page

For contact page I used flexbox and planned to use my contact details in left section of the page and provide form that will launch user email program at the right side of the screen where visitor can enter their name, email with some query that they have which can be further mailed using submit button.
